# TRON 波场靓号生成器 | TRON Vanity Address Generator

一个高效的 Rust 实现的 TRON 波场靓号地址生成器，支持快速生成和识别靓号，输出完整的私钥、公钥、地址和 BIP39 助记词。

## 功能特性 ✨

- **⚡ 超高性能**: 充分利用 CPU 多核并行处理，支持 GPU 加速潜力
- **🎨 简洁 UI 界面**: 彩色控制台输出，实时显示生成进度和统计信息
- **🔍 智能靓号识别**: 支持多种靓号检测规则：
  - 连续相同字符（如 `AAAA`, `1111`）
  - 递增/递减序列（如 `1234`, `DCBA`）
  - 自定义模式匹配
- **📝 完整信息**: 每个靓号包含：
  - TRON 地址 (T 开头)
  - 私钥 (128 位十六进制)
  - 公钥 (非压缩格式)
  - BIP39 助记词 (12 个单词)
- **💾 双重保存**:
  - 实时输出到控制台（彩色高亮）
  - 持久化保存到文件（tron_vanity.txt）
- **⚙️ 灵活配置**:
  - 自定义靓号模式
  - 调整线程数和批处理大小
  - 可选保存所有地址

## 系统要求

## 靓号检测规则

程序会自动识别以下类型的靓号：

### 1. 连续相同字符（≥3 个）

- `Taaa...` - 3 个'a'
- `T1111...` - 4 个'1'
- `TBBBB...` - 4 个'B'

### 2. 递增序列

- `T123...` - 递增数字
- `TABC...` - 递增字母

### 3. 递减序列

- `T321...` - 递减数字
- `TCBA...` - 递减字母

### 4. 自定义模式

用户可输入任何想要的子串。

## TRON 地址生成算法

1. **生成私钥**: 随机 32 字节
2. **生成公钥**: 使用 Secp256k1 曲线从私钥推导公钥
3. **地址生成**:
   - SHA256(公钥)
   - RIPEMD160(结果)
   - 添加版本字节 (0x41 for TRON)
   - 计算校验和 (前 4 字节的 SHA256(SHA256))
   - Base58 编码
4. **生成助记词**: 使用前 16 字节私钥生成 12 个单词

## 常见问题

### Q: 如何确保生成的地址有效？

A: 所有地址都使用标准 TRON 算法生成，并包含校验和验证。可以在 TRON 浏览器验证。

### Q: 找到靓号需要多久？

A: 取决于硬件和靓号复杂度。3 个连续字符的靓号平均需要 5-10 分钟（8 核 CPU）。

### Q: 生成的私钥是否安全？

A: 使用系统随机数生成器。对于生产用途，建议在离线环境中使用。

### Q: 可以用 GPU 加速吗？

A: 当前版本使用 CPU 多线程。GPU 加速可作为未来优化项。

### Q: 如何导入生成的地址到钱包？

A: 使用私钥（Hex 格式）或助记词（BIP39 格式）导入到支持 TRON 的钱包。
